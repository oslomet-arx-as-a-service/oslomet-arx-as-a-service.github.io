Anonymization-as-a-Service

Description

Our goal is to develop and build a data anonymization service along with a software framework for simple user interaction. The solution will provide availability to the already existing anonymization functionality from the open-source ARX software library. The technical delivery is separated into two logical components; a web-API service that will deliver ARX functionality through a well documented RESTful API, and one or more API consumer modules that will allow integration with data-science tools. To the author's knowledge, this kind of data-anonymization-as-a-service product does not exist at the present time.
